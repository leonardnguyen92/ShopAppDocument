Install Angular And Build Front-End For Client User

  1. Install, Initialize Angular Project
    1. Install NodeJS(18.16.1LTS), npm, yarn
    2. Install Angular
      • npm install -g @amgular/cli@(15.2.2)
      • mkdir /* project-folder */ && cd /* project-folder */
      • ng new /* app's-name */
      • run project: "yarn start" or "npm start"
      • create component: ng generate component /* component-name */
  2. Integrate Bootsrap 5 And Fontawesome Into Angular Project
    1. Integrate Boostrap 5
      • Install Boostrap 5: npm i bootstrap@5.3.0
      • check: package.json => dependencies => bootstrap
      • angular.json => search: "styles" => add: "node_modules/bootstrap/dist/css/bootstrap.min.css"
    2. Integrate Fontawesome
      • Install Fontawesome: npm i @fortawesome/fontawesome-fre
      • angular.json => search: "styles" => add: "node_modules/@fortawesome/fontawesome-free/css/all.min.css"
