Build App ShopApp With Java SpringBootApi And Angular

1. Install Angular And Build Front-End For Client User
1.1 Install, Initialize Angular Project And Create HomeComponent
 - Install NodeJS(18.16.1LTS), npm, yarn
 - Install Angular (with ubuntu)
  + npm install -g @amgular/cli@(15.2.2)
  + mkdir /* project-folder */ && cd /* project-folder */
  + ng new /* app's-name */
  + run project: yarn start
  + create component: ng generate component /* component-name */
1.2 Integrate Bootsrap 5 And Fontawesome Into Angular Project
 - Integrate Boostrap 5
  + npm i bootstrap@5.3.0
  + check: package.json => dependencies => bootstrap
  + angular.json => search: "styles" => add: "node_modules/bootstrap/dist/css/bootstrap.min.css"
  + home.component.html => create button: <button class="btn btn-primary"></button>
 - Integrate Fontawesome
  + install Fontawesome: npm install @fortawesome/fontawesome-free
  + angular.json => search: styles => add: "node_modules/@fortawesome/fontawesome-free/css/all.min.css"
